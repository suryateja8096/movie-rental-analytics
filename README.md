🎬 SAKILA DVD RENTAL ANALYSIS PROJECT
📌 OVERVIEW

This project presents a comprehensive analysis of the Sakila DVD Rental business using SQL and Power BI. The goal is to uncover key insights related to customer behavior, rental trends, revenue generation, and inventory performance to support data-driven decision-making.

🎯 OBJECTIVES
Analyze rental patterns and trends
Identify high-performing film categories
Evaluate customer engagement and segmentation
Compare store performance
Optimize inventory and stock management
🛠️ TOOLS & TECHNOLOGIES
SQL (MySQL) – Data extraction and transformation
Power BI – Dashboard creation and visualization
Excel – Data preprocessing (if applicable)

🔍 KEY INSIGHTS
Rentals show seasonal trends, peaking mid-period
Top categories like Action and Sports drive most revenue
Customer base is globally distributed
High customer retention with majority active users
Stores show balanced performance
Inventory optimization is needed for high-demand films
📁 DATABASE

This project uses the Sakila Database, a sample database designed to simulate a DVD rental store.

🚀 HOW TO USE
Import the Sakila database into MySQL
Run SQL queries to extract insights
Connect Power BI to the database
Build visuals and dashboards
Analyze and interpret results
